# Uptime Kuma — Configuración inicial

## Primera vez
1. Entrar a https://status.kraftdo.cl después de `docker compose up -d`
2. Crear usuario admin

## Monitores esenciales
- KraftDo Admin: https://kraftdo.cl/admin
- API Python: https://api.kraftdo.cl/health
- Upload Portal: https://sistema.kraftdo.cl/health
- Classifier: https://app.kraftdo.cl
- n8n: https://n8n.kraftdo.cl
- MySQL (Docker container): kraftdo_mysql
- Redis (Docker container): kraftdo_redis

## Alertas WhatsApp
